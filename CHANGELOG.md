# Changelog / 更新日志

All notable changes to this project are documented here.

本文件记录本项目的重要变更。

## [0.1.0] - 2026-07-15

### English

- Initial public release of the Windows Codex and ChatGPT desktop plugin.
- Added Baidu OAuth setup, authorization status, account information, directory listing, recursive search, and file download tools.
- Added Windows DPAPI encryption for the App Secret and OAuth tokens.
- Added bilingual English and Chinese documentation and plugin metadata.
- Added an installable release ZIP, SHA-256 checksum, and automated release workflow.

### 中文

- 首次公开发布适用于 Windows 的 Codex 和 ChatGPT 桌面版插件。
- 提供百度 OAuth 设置、授权状态、账号信息、目录列表、递归搜索和文件下载工具。
- 使用 Windows DPAPI 加密应用 Secret Key 和 OAuth 令牌。
- 添加中英双语文档和插件界面元数据。
- 添加可安装的 Release ZIP、SHA-256 校验文件和自动发布工作流。
