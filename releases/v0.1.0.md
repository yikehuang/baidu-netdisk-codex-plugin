# Baidu Netdisk Plugin v0.1.0 / 百度网盘插件 v0.1.0

## English

This is the first public release of the unofficial Windows-only Baidu Netdisk plugin for Codex and the ChatGPT desktop app.

### Included

- Official Baidu OAuth and XPan API integration.
- Authorization status and account information tools.
- Directory listing and recursive file search.
- Downloads by `fs_id`, with local filename sanitization.
- Windows DPAPI encryption for the App Secret and OAuth tokens.
- English and Chinese documentation and plugin metadata.

Download `baidu-netdisk-codex-plugin-v0.1.0.zip` and verify it against `baidu-netdisk-codex-plugin-v0.1.0.zip.sha256` before installation.

### Scope

The plugin only reads and downloads files. It does not upload, delete, move, rename, publish, or share files. It does not bypass Baidu share pages, extraction codes, CAPTCHA, membership restrictions, rate limits, or safety checks.

## 中文

这是非官方 Windows 百度网盘 Codex 与 ChatGPT 桌面版插件的首个公开版本。

### 包含内容

- 接入百度官方 OAuth 与 XPan API。
- 提供授权状态和账号信息工具。
- 提供目录列表和递归文件搜索。
- 按 `fs_id` 下载文件，并清理本地文件名中的非法字符。
- 使用 Windows DPAPI 加密应用 Secret Key 和 OAuth 令牌。
- 提供中英双语文档和插件界面元数据。

安装前请下载 `baidu-netdisk-codex-plugin-v0.1.0.zip`，并使用 `baidu-netdisk-codex-plugin-v0.1.0.zip.sha256` 核对文件完整性。

### 功能边界

插件只读取和下载文件，不会上传、删除、移动、重命名、公开或分享文件。它不会绕过百度分享页、提取码、验证码、会员限制、频率限制或安全检查。
