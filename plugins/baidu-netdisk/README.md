# Baidu Netdisk / 百度网盘

## English

This Windows-only Codex and ChatGPT desktop plugin lists, searches, and downloads files from an authorized Baidu Netdisk account through the official OAuth API.

Run `baidu_open_oauth_setup` after installation to configure the local OAuth connection. Secrets and tokens are encrypted for the current Windows user with DPAPI and are never stored in this plugin directory.

The plugin is read-and-download only. It does not upload, delete, move, rename, publish, or share files, and it does not bypass Baidu share pages or access restrictions.

## 中文

这是一个仅支持 Windows 的 Codex 和 ChatGPT 桌面版插件。它通过百度官方 OAuth API，列出、搜索并下载已授权百度网盘账号中的文件。

安装后运行 `baidu_open_oauth_setup`，配置本地 OAuth 连接。Secret Key 和令牌会使用 DPAPI 为当前 Windows 用户加密，绝不会保存在插件目录中。

插件只提供读取和下载功能，不会上传、删除、移动、重命名、公开或分享文件，也不会绕过百度分享页或访问限制。

See the [repository README](../../README.md) for full installation, authorization, tools, and security details.

完整的安装、授权、工具和安全说明见[仓库 README](../../README.md)。
